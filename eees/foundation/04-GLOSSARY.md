# Glosario

- ERP: aplicacion interna de gestion operacional, RRHH, reclutamiento, BI y documentos.
- EEES: Enterprise ERP Engineering Standard.
- Guardian: validador ejecutable de reglas EEES.
- RPC: funcion PostgreSQL expuesta via Supabase.
- RLS: Row Level Security.
- BUK: sistema externo fuente de verdad para trabajadores, cargos, areas y documentos laborales.
- Folio: requerimiento operativo de contratacion.
- Who: etapa de aprobacion de candidato dentro de reclutamiento.
- Capability: permiso fino complementario a roles y modulos.
- Baseline: fotografia auditada que congela deuda conocida y bloquea regresiones nuevas.
