# AGENTS.md

Este repositorio usa EEES como estandar tecnico ejecutable.

Antes de trabajos no triviales, lee `eees/codex/BOOT_SEQUENCE.md`, `tasks/todo.md` y `tasks/lessons.md`.

Reglas operativas:

- Planifica en `tasks/todo.md` cambios de 3 o mas pasos.
- Inspecciona contratos reales antes de editar.
- Usa `npm run guardian` para cambios enterprise.
- No relajes RLS, grants ni permisos para hacer pasar UI.
- Registra lecciones cuando una correccion del usuario revele un patron repetible.
